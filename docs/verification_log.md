### H2. 회전율 — 재구현이 아니라 '되찾기'

목표비중은 점수·가격만의 함수라 **포트폴리오 금액과 무관**하다. 따라서 비용률만 0 으로 둔
대조 백테스트와 자산곡선 비율을 내면 엔진이 실제로 청구한 비용 계수 Π(1−cost×turnover) 만
남고, 여기서 회전율을 정확히 되찾을 수 있다(`analysis/turnover.py`). 밖에서 비중을 다시
계산하는 추정은 엔진과 어긋날 수 있어 택하지 않았다(`exposure.py` 와 같은 원칙).
전제가 깨지면(비용비율 상승·회전율>1) 조용히 틀린 값을 내는 대신 **멈춘다**.

두 계층의 비용을 분리해 끄기 위해 상위 리밸런싱 측정에서는 **슬리브 비용을 양쪽 다 켠 채**
상위 비용만 껐다. 함께 끄면 슬리브 수익이 달라져 ±7%p 임계 트리거 날짜가 어긋나고,
'비용만 다른 같은 경로'라는 전제가 무너지기 때문이다.

| 계층 | 회전 횟수 | 총회전율%(포트폴리오 기준) | 연평균% | 연 비용드래그%p |
|---|---|---|---|---|
| 슬리브 로테이션(월간) | 77 | 2,037 | 339.6 | 0.340 |
| 상위 리밸런싱(분기+임계) | 25 | 28 | 4.7 | 0.005 |
| **합계** | — | — | **344.3** | **0.344** |

연도별(포트폴리오 기준 회전율%): 2020 264 · 2021 381 · 2022 **159**(최저) · 2023 261 ·
2024 **395**(최고) · 2025 381 · 2026(반기) 225. `reports/turnover_by_year.csv`.

해석 주의 2건: ① 회전율은 **단방향**(100% = 포트폴리오를 한 번 갈아엎음). ② 슬리브 회전
77회는 체크 횟수와 같다 — 종목 '교체'(75회)뿐 아니라 매달 목표비중 복원(드리프트)과 부분
충전율 변화가 함께 들어간다. ③ 드래그 0.34%p/년은 **이미 백테스트 수익률에 반영된** 값이므로
성과에서 추가로 빼면 이중 차감이다.

### H3. 국면별 성과 분해 (KOSPI200 200MA)

국면은 KOSPI200 종가 vs 200일 이동평균 **+ 20일 기울기**로 나눈다. 수준만 쓰면 MA 를 스치는
구간이 상승/하락으로 튀어 국면이 잘게 부서지므로, 방향이 확인된 구간만 상승·하락으로 부르고
애매한 구간은 횡보로 모은다. **사후 라벨이지 매매 신호가 아니다** — 전략은 이 라벨을 보지
않으므로(전략 입력은 종목별 TrendScore 뿐) 같은 날 종가로 라벨을 붙여도 룩어헤드가 성립하지 않는다.

| 국면(일수·비중) | 대상 | 누적수익% | 연율화% | MDD% | V2 평균노출% |
|---|---|---|---|---|---|
| 상승 (901일·57.5%) | HELM(동결 V2) | +108.66 | 22.84 | −12.71 | **59.2** |
| | V1 기준선(이진 60) | +114.63 | 23.81 | −12.71 | |
| | KODEX TRF7030 | +107.33 | 22.62 | −8.44 | |
| 하락 (444일·28.4%) | HELM(동결 V2) | **+4.50** | 2.53 | **−10.03** | **34.7** |
| | V1 기준선(이진 60) | **+10.39** | 5.77 | −11.92 | |
| | KODEX TRF7030 | +2.81 | 1.59 | **−18.54** | |
| 횡보 (221일·14.1%) | HELM(동결 V2) | **+3.46** | 3.96 | −11.70 | **42.4** |
| | V1 기준선(이진 60) | **−6.70** | −7.61 | −15.50 | |
| | KODEX TRF7030 | +4.41 | 5.05 | −15.30 | |

읽는 법 3가지:
1. **하락장 방어는 벤치 대비 확인** — 수익 +4.5 vs +2.8, MDD −10.0 vs −18.5. 평균 노출이
   만충 70% 대비 34.7% 로 내려간 것이 그 인과다(게이트가 실제로 셔터를 내렸다).
2. **횡보장에서 V2 +3.5 vs V1 −6.7** — Tier 2-a 가 겨냥한 '전환 구간'이 국면 축에서도
   개선으로 확인된다. 구간 상태별 손익의 5슬롯대 부호 반전(−1.08 → +2.83)과 같은 현상을
   다른 축에서 본 것이다.
3. **불리한 사실: 하락장은 V1 이 더 방어적**(+10.4 vs +4.5). 부분 충전이 하락장에 노출을
   남기는 것은 Tier 2-a 의 **명시적 비용**이며 동결 판정에 이미 반영돼 있다(그 대가로 횡보
   +10.2%p·최저 해 +0.2→+2.2). 발표에서 먼저 말할 항목.

정의 주의: 국면 MDD 는 **그 국면 구간만 이어 붙인 합성곡선** 기준이라 국면 사이의 회복이
빠져 있다. 전체 구간 MDD 와 직접 비교하지 않는다. 상승 국면 MDD 가 전체 MDD 와 같은
−12.71 인 것은 우연이 아니라 사실 확인 결과다 — 최대낙폭 구간(2026-02-26 고점 → 03-04 저점,
4거래일)이 통째로 상승 국면 안에 있다(별도 확인).

### H4. 독스트링 정합 — `entry_gate` 권고(60) vs 동결값(52)

`satellite/backtester_v2.py` 의 `entry_gate` 설명이 "크기만 조절하고 싶으면 60 으로 고정"
으로 끝나 있어, 동결 구성(문턱 52)과 어긋난 채로 읽혔다. 코드를 처음 보는 팀원이 동결값을
설정 실수로 오해할 수 있는 자리다. 한 문단을 덧붙여 해소했다(**독스트링만 수정 · 동작 무변경**):
권고는 '**분리 측정할 때** 크기와 자격을 섞지 말라'는 실험 규율이고, 채택값 52 는 그 분리
측정을 마친 뒤 문턱 48~52 그리드 plateau 검증으로 고른 값이다(문턱 60 고정 구성은 전체 구간
Calmar 0.90 으로 기각). 이는 '기록된 실수 3(전달 체인 오염)'과 같은 종류의 잔존물이었다 —
설계 제안 문구가 측정 결과 위에 남아 있던 경우.

### H5. 조립 일원화 — `analysis/frozen.py`

분석 러너가 6개로 늘며 동결 구성 조립이 러너마다 복붙됐다. 한쪽만 바뀌면 **서로 다른 상품의
수치가 같은 제안서에** 실린다(이 프로젝트에서 조용한 이탈은 이미 여러 번 사고를 냈다).
`build_sleeve()`·`build_irp()` 로 모았고, 동결값 자체는 `run_v2.FROZEN_RAMP` 에 그대로 뒀다
(복제하면 동결값이 두 곳에 생긴다). 신규 러너 3종이 이를 사용하며, 기존 러너는 무수정.

## 한계 · 미검증

정직하게 남긴다. 아래는 확인하지 **않은** 것들이다.

- **롤링 표본의 독립성 없음** — 백테스트가 78개월뿐이라 보유 60개월 창은 18개고 서로 겹친다.
  손실 확률은 "이 6.5년 안에서 시작 시점을 굴렸을 때"의 값이지 미래 확률 추정치가 아니다.
- **10년 시나리오 미산출** — 데이터가 6.5년뿐이라 만들 수 없다. 만들었다면 조작이다.
- **자동화된 테스트 스위트 없음** — 위 검증은 전부 일회성 스크립트다. `pytest` 로 고정해
  두지 않아 회귀를 자동으로 잡지 못한다. A1(축퇴 일치)·D1~D5(적립식 수학)는 테스트로
  박아 둘 가치가 있다.
- **거래비용 민감도 미측정** — 회전율과 그 드래그는 이제 측정돼 있으나(H2: 연 344%·0.34%p),
  비용률 0.10% **가정 자체**를 흔들어 보지는 않았다. 드래그가 회전율에 선형이므로 비용이
  2배면 드래그도 2배(0.69%p/년)라는 산술 추정은 가능하지만, 그 경우 성과 순위가 뒤집히는지는
  재보지 않았다(비용 변경은 파라미터 실험이 아니므로 동결 위반은 아니다 — 미실행일 뿐).
- **국면 라벨의 파편화** — 200MA+기울기 규칙은 전체 구간을 43개 블록으로 나누고, 그중 일부는
  1~3일짜리다. 국면별 **집계**(일수 기준 평균)에는 문제가 없으나, "국면이 몇 번 바뀌었나"
  같은 블록 단위 통계로는 쓸 수 없다. 라벨 규칙(200/20)의 주변값 강건성도 재지 않았다.
- **상대지표의 구간 의존성** — IR·TE 는 전체·2025컷 두 구간 값만 있다. 롤링 IR(예 12개월
  창)의 안정성은 재지 않았으므로, "IR 이 음수다"는 이 두 창에서의 사실이지 시간에 걸친
  일관된 성질이라고 주장하지 않는다.
- **슬리피지·유동성 미반영** — 한국 섹터 ETF 일부는 호가가 얇다. 백테스트는 종가 체결 가정.
- **세금 미반영** — IRP 는 과세이연이라 일반계좌와 다르나, 인출 시 연금소득세가 있다.
  적립식 분석은 세전 기준이다.
