  값). 판정 재료로 쓸 수 없음.
- **B-1·C-1**: MDD 를 −9%대로 확실히 축소하나 **컷 Calmar 가 기준선(1.086) 밑으로** 내려감
  (B-1 10%=0.957, C-1=0.961) → 유지축소·reversal 과 같은 **국면 편중** 패턴.
- **C-2**: 컷은 1.105 로 올리나 전체 Calmar 0.999 로 기준선 미달 → 역시 국면 편중.
- **C-3 지역 집중 상한 — 그룹핑 granularity 비교(세부 4그룹 vs 광역 3지역)**:

### A-2 iso 방향성 검증 — iso 의 4/4 는 '듀레이션 로테이션'인가

A-2 **성과** 측정은 spec(439870)도 전체 창(2020~)이다 — 스위칭이 439870 의 상장 늦음(2022-08)을
흡수하므로. 다만 **439870 자체의 방향성 지표**(60일 가격변화)는 상장 후만 잴 수 있어 spec 으론
방향성 검증이 절반뿐이다. 반면 **iso 3채권(153130 단기채·114260 국고3·273130 종합채권)은 모두

- 단서: 273130 은 **종합채권(AA-이상 크레딧 액티브·중기 듀레이션)**이라 순수 장기국채보다 금리
  민감도가 낮아 방향성이 국고3(2.4×)보다 무디다(편입률 1.6×). 방향은 확실하나 스프레드는 좁다.
  단기채(현금 대용)는 편입 상시 100%라 방향 신호 없음(예상대로).

제3안(넓은 듀레이션 사다리 + 절단 없는 측정)을 줄 종목이 **실재하는지** 국내 상장 목록을 웹

471230 은 2023 상장, KBSTAR·KODEX 30년물은 전부 2021~ 상장, 국채선물 ETF 는 파생이라 IRP 불가 —
